Methods

Listen(selector)

listen to a specific Property or a complete State change you can use the SafeBehaviorSubject same as a BehaviorSubject but follow Functions has no effect next, complete, error
Parameters:
Name Type Description
selector select the Part of the State to Listen
Example
// write the data Property of the State into the console
store.Listen(s => s.data).subscribe(d => console.info(d));

Mutate(selector, mutation)

mutate a specific Property or a complete State
Parameters:
Name Type Description
selector select the Part of the State to Mutate
mutation define how to change the State
Examples
// simple Example to set the State
store.Mutate(s => s.data, () => newData);
// make a complex Mutation
store.Mutate(s => s, old => {
    old.data = [];
    old.loading = true;
    return old;
});

parseSelectorAccess(selector)

create a unique Key from the State Selector that can use as identifier for the Behavior Subjects adds "root" in front of the Key to have no empty Key when selector points to the State itself
Parameters:
Name Type Description
selector
Example
store.Listen(s => s.data) // generates the key: root.data
store.Listen(s => s) // generates the key: root
